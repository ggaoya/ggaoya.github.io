hexo命令

发表于: 2017.11.09 | 分类： -

hexo

npm install hexo -g #安装
npm update hexo -g #升级
hexo init #初始化

简写

hexo n “新建文章” => hexo new “新建文章” #新建文章
hexo p => hexo publish #草稿发布
hexo g => hexo generate #生成静态文件
hexo s => hexo server #启动服务
hexo d => hexo degploy #部署

服务器

hexo s #Hexo 会监视文件变动并自动更新，您无须重启服务器。
hexo s -s #静态模式
hexo -p 5000 #更改默认端口
hexo s -i 192.168.0.1 #更改默认ip

hexo clean #清除缓存

hexo g –watch #监听文件变动

模版

hexo new “创建文章”
hexo new page “创建页面”
hexo generate #生成静态页面至pulic目录
hexo server #开启预览访问端口
hexo deploy #将.deploy目录部署到GitHub
